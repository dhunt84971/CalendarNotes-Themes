Create a PRD to describe an application with the following features.

The application will be a desktop application written in nodejs using electron, targeting both Linux and Windows platforms.  This application, called Calendar Notes Themes, will be used to create and edit Calendar Notes theme files for use with the Calendar Notes application whose source code can be found in the /home/dave/gitWorkspace/CalendarNotes folder.  The application needs to allow the user to create a new theme file using the default theme as the starting point or open an existing theme file.  When a theme file is opened or created the application will display an preview of the application interface and allow the user to select different parts of the application interface by clicking on them with the mouse.  When clicked upon, if the object that was clicked has a style that is settable within the defined theme styles, the associated style setting variable name(s) and current value(s) will be displayed.  The user will then have the ability to change these values.  In the case of a style that specifies a color, the user will have the option of displaying the color pallette selector for choosing a color or they can enter the color code or name directly.  When the user has finished making changes an Apply button will be available for the user to apply the changes to the preview of the application.  A Cancel button will also be available for canceling any changes.  The user will have the ability to save any edits to the opened file, or Save As to a different file.  Changes will not be made to the theme file until the user has saved the changes.

This application will start as a stand-a-lone application, but will later be integrated into the Calendar Notes application, allowing the user to launch the theme editor directly from Calendar Notes.

Create a README.md file describing the application and include a section for the developer which describe the dependencies and how to build the installable package.  Use the same packaging as was implemented for Calendar Notes.
